# Alma Testing

Tools for testing and controlled record movement workflows in Alma.

## Tools
- [Move Authority Prod to Sandbox](Move_Authority_Prod_to_Sandbox/) - Copies an authority record from Alma Production to Alma Sandbox
- [Move Authority Sandbox to Prod](Move_Authority_Sandbox_to_Prod/) - Copies an authority record from Alma Sandbox to Alma Production
- [Move Bib Prod to Sandbox](Move_Bib_Prod_to_Sandbox/) - Copies a bibliographic record and related inventory from Alma Production to Alma Sandbox
