# Alma Automation

Tools for automating tasks in Alma.

## Tools

- [Getting Started](Getting%20Started/) - Onboarding path with Local Testing before larger automation workflows
- [Ceased Title](Ceased%20Title/) - Automates ceased title processing
- [ISSN Update](ISSN%20Update/) - Automates updating ISSN fields in Alma bibliographic records
- [Title Change](Title%20Change/) - Automates catalog record updates when a serial title undergoes a title change
- [URL Redirect](URL%20Redirect/) - Converts Alma video record URL patterns and records final redirected URLs in XLSX
